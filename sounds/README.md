# Ses Dosyaları

Bu klasörlere ses dosyalarınızı ekleyin:

## Klasörler

- **ziller/**: Zil sesleri (MP3 formatında)
  - Örnek: `zil1.mp3`, `zil2.mp3`
  
- **marslar/**: İstiklal Marşı ve diğer marşlar
  - Örnek: `istiklal.mp3`
  
- **siren/**: Siren sesleri
  - Örnek: `siren.mp3`
  
- **muzik_yayini/**: Teneffüs müzik yayını (isteğe bağlı)
  - Rastgele çalınacak MP3 dosyaları

## Not

- Ses dosyaları MP3 formatında olmalıdır
- Dosya adları Türkçe karakter içerebilir
- Program ses dosyası bulamazsa hata loglanır ancak çalışmaya devam eder
